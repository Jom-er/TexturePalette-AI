# TexturePalette AI

TexturePalette AI is a tool that analyzes Minecraft-style texture images to extract and interpret their dominant colors.

The application combines local image processing with cloud-based AI analysis to provide more accurate color identification. It first performs fast, client-side color clustering using the K-Means algorithm to generate immediate results. It then enhances the output using an AI model to infer the most likely base material color, reducing the impact of shadows, lighting variations, and pixel-art noise.

---

## Key Features

- Instant dominant color extraction using K-Means clustering  
- AI-assisted color interpretation for improved accuracy  
- Focus on underlying material color rather than lighting effects  
- Optimized for pixel-art and game texture analysis  
- Fast, lightweight, and responsive design  

---

## Purpose

This project is designed to help developers, artists, and mod creators better understand and standardize texture color palettes for game assets and creative workflows.
